FluvialGeomorphologyToolbox

A toolbox to extract data organized along a spatial continuum.

Download this project as a .zip file Download this project as a tar.gz file

The ArcGIS Fluvial Geomorphology toolbox comprises 15 Python scripts including ‘Create Transects, ‘Moving Window’, ‘Local Moving Window’ and ‘Distance Along Line (DAL)’ that each generate respective sets of geometrical or tabular outputs to analyse spatial continuums.

This toolbox was presented at the 2014 Northwest GIS User Conference, in Lynnwood, WA.

9. Points Inside Polygon
10. Polygon to Points
11. RasterSubstraction
12. Distance along Line (DAL)
13. SelectionTool
14. ShoreMaker
15. TransectsDAL

Download the toolbox and add it to your map in ArcMap 10.X. Check the 'READ_ME_PythonCodeInstructions.txt' file for more information. Contact me if you have any questions.
